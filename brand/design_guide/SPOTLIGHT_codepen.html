Spotlight Effect

✦
Ambient Illumination
Two blurred orbs sit behind each card and follow the cursor. The effect is diffused light, not a sharp highlight.
Design
⬡
CSS Custom Properties
Mouse coordinates are written as --mouse-x and --mouse-y on each card via JavaScript.
CSS
◎
Pseudo-elements
No extra DOM nodes needed. The ::before and ::after pseudo-elements carry the entire glow effect.
Performance
▲
Rim Light Detail
The 1px wrapper padding creates a hairline gap. The glow bleeds into it, forming a colored rim light at the card edge nearest the cursor.
Detail
◈
Per-card Tracking
Each card independently converts the global cursor position into local coordinates, so every card reacts simultaneously.
JavaScript
⬨
Backdrop Blur
The semi-transparent card background with backdrop-filter lets the glow bleed through from behind, deepening the lighting illusion.
CSS
